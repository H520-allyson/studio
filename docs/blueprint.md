# **App Name**: Print Genie

## Core Features:

- Landing Page: Displays key printing services and company information.
- AI Print Assistant: Gemini API-integrated chatbot answering client questions about printing materials, finishes, and pricing.
- Interactive Cost Calculator: Dynamic form estimating prices in real-time based on product type, size, quantity, and finish.
- Advanced Order/Upload Form: Collects client details, instructions, and print files with real-time progress bar and resolution pre-check.
- Order Status Tracking: Page displaying the current status of an order, accessible via a unique Order Reference Number.
- Admin Order Dashboard: Secure admin panel displaying and allowing management of all orders from the Firestore database.
- Firebase Integration: Secure Firebase authentication for the admin panel. Utilize Firestore to manage/retrieve data and Storage for file uploads.

## Style Guidelines:

- Primary color: Neon green (#39FF14) to highlight interactive elements and actions.
- Background color: Dark gray (#222222) to support a modern dark-mode theme.
- Accent color: Electric blue (#7DF9FF) to create contrast and focus on secondary interactive elements.
- Body and headline font: 'Inter' for a modern, neutral feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use Material Design 3 icons throughout the application.
- Fully responsive layout for desktop and mobile browsers.
- Subtle animations and transitions to enhance user experience (e.g., loading states, progress bars).
